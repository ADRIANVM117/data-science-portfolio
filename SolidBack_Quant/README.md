¿Que intenta capturar esta estrategia?
"Cuando dos acciones muy correlacionadas del mismo sector divergen temporalmente, la acción más pequeña (follower) tenderá a ponerse al día con la acción más grande (leader)." 

1. No es una estrategia de momentum.

2. No es una estrategia direccional.

3. No es una estrategia de market making.

Es una especie de:
  - Relative Value
  - Lead-Lag
  - Statistical Arbitrage simplificado
